# Hlink 测试指南

## 当前功能支持情况

| 功能 | 桌面端 | 移动端 | 说明 |
|------|--------|--------|------|
| 文本同步 | ✅ | ✅ | 完全支持 |
| 图片同步 | ✅ | ✅ | 桌面端和移动端都支持 |
| 文件同步 | ❌ | ❌ | 未实现 |

---

## 测试环境准备

### 1. 启动桌面端
```bash
npm run tauri dev
```

### 2. 启动移动端（Android）
```bash
npm run android:dev
```

### 3. 确保设备连接
- 桌面端和移动端连接到**同一 Wi-Fi 网络**
- 在移动端扫描桌面端二维码或手动输入 IP:端口连接

---

## 测试步骤

### ✅ 测试 1：文本同步（桌面 → 移动）

**步骤：**
1. 在桌面端复制一段文字（例如：`Hello from Desktop!`）
2. 观察桌面端同步历史，应该显示 `[文本]` 记录
3. 切换到移动端应用
4. 在移动端任意输入框粘贴，应该能看到复制的文字

**预期结果：**
- ✅ 桌面端显示同步历史
- ✅ 移动端剪贴板包含复制的文字
- ✅ 移动端显示 Toast 提示 "📋 已同步到剪贴板"

---

### ✅ 测试 2：文本同步（移动 → 桌面）

**步骤：**
1. 在移动端任意应用复制文字（例如：`Hello from Mobile!`）
2. 切换到 Hlink 应用（保持在前台）
3. 观察移动端同步历史
4. 在桌面端任意应用粘贴

**预期结果：**
- ✅ 移动端显示同步历史
- ✅ 桌面端剪贴板包含复制的文字
- ✅ 桌面端可以直接粘贴

---

### 🖼️ 测试 3：图片同步（桌面 → 移动）

**当前状态：** ⚠️ 仅桌面端支持

**步骤：**
1. 在桌面端复制一张图片（截图、从浏览器复制图片等）
2. 观察桌面端同步历史，应该显示 `[图片]` 记录
3. 切换到移动端应用
4. 在移动端任意支持图片的应用中粘贴

**预期结果：**
- ✅ 桌面端显示 `[图片]` 同步历史
- ✅ 移动端接收图片并写入剪贴板
- ✅ 移动端显示 Toast 提示 "🖼️ 图片已同步到剪贴板"
- ✅ 移动端可以在其他应用中粘贴图片

**注意：** 
- 移动端使用 Web Clipboard API 写入图片
- 如果 Web API 不支持，会降级到 Tauri 命令（桌面端）
- 如果都不支持，会显示提示信息

---

### 🖼️ 测试 4：图片同步（移动 → 桌面）

**步骤：**

2. 切换到 Hlink 应用
3. 观察移动端同步历史

**预期结果：**
- ❌ 移动端**目前不支持**图片读取（Android 剪贴板 API 限制）

---

## 调试技巧

### 查看日志

**桌面端：**
- 查看终端输出的 Rust 日志
- 使用浏览器开发者工具查看前端日志

**移动端：**
```bash
# 查看 Android 日志
adb logcat | grep -i hlink

# 或查看所有日志
adb logcat
```

### 检查连接状态

1. 查看设备列表：应该显示已连接的设备
2. 查看同步历史：应该显示同步记录
3. 检查网络：确保在同一 Wi-Fi 网络

### 常见问题

**Q: 移动端收不到桌面端的同步？**
- 检查移动端应用是否在前台（Android 10+ 限制）
- 检查设备是否已连接（查看设备列表）
- 检查同步开关是否开启

**Q: 桌面端收不到移动端的同步？**
- 检查移动端应用是否在前台
- 检查移动端同步开关是否开启
- 查看移动端日志是否有错误

**Q: 图片同步不工作？**
- 桌面端：✅ 支持
- 移动端：✅ 已支持（使用 Web Clipboard API）
- 如果移动端无法写入，检查浏览器/WebView 是否支持 Clipboard API

## 改进建议

### 1. ✅ 移动端图片支持（已完成）

已实现移动端图片同步功能：
- 使用 Web Clipboard API 写入图片
- 降级到 Tauri 命令（桌面端）
- 错误处理和用户提示

### 2. 移动端图片读取（待实现）

当前移动端无法读取剪贴板中的图片（Android 限制）：
- Android 剪贴板 API 主要支持文本
- 需要特殊权限或 Accessibility Service
- 可以考虑使用文件选择器作为替代方案

### 2. 实现文件传输

需要：
- 文件分块传输协议
- 文件元数据（文件名、大小、类型）
- 传输进度显示
- 错误处理和重试机制
